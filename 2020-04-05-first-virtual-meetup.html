First virtual meetup.

Posted by Dmytro Kurbakov on April 5, 2020

New home.

Corona virus outbracke significantly influenced our daily way of life. We should adapt to the new reality and new way of working.

We value all participants and health is the priority number one. So we decided to move all our metups to virtual meetups. I want to thay thank you to everyone who putted all effords to make in happen.


Currently we will have a virtual meetup every Sunday, at 12am. We decided to keep working and move towards our goal: join Robocup@Work 2021.

To make it happen we created the simulation environment in Gazebo, where you can already see the robot, lidar, stereo camera, odometry and IMU. It is all what we need to work on map generation and localization in real time. We currently working on improvement the simulation and will keep working on simulation of the arena of the Robocup arena.

Whant to try it out? See our repository for simulation

.

We also addressed the problem of the varity of working enviriments on each feveloper machines. So we have a docker image that you can use to make sure you do not waste your time of setting up the ROS and all dependencies.

Chek it our and let us know if you need anything else. Docker image can be found here.


Stay safe and see you soon!
